About BITS

For collaborative issue tracking.

Bug Issue Tracker Solutions provides an easy-to-use web-based application tool to help individuals and businesses track issues and find lasting solutions to their needs. It is driven by a vibrant user community ensuring collobaration on all levels.

Bug Issue Tracker Solutions provides you with a central place to submit and comment on bugs and features. The focus is on accountability and progress, ensuring that not only does the work get done, but it gets done correctly.

The user community decides the priority through upvotes. The user community can also participate in the review process through commenting, ensuring that issues are dealt with correctly. In effect the user community acts as a second set of eyes.

Every issue is tracked from inception to completion. Each issue is assigned to an assignee best suited to the issue in order to encourage forward progress and reduce miscommunication. This organized approach to managing issues provides many valuable insights that can be used to refine and improve the system and deal with future issues more effectively.

Key features:

Web-based Architecture

Uses Django, a high-level Python Web framework that ensures speed, security and ease of use. All features and functionality are available to the user with a Web browser. Cloud based with nothing to install.

Standard Database Support

Uses postgresql, a powerful, open source object-relational database system that uses and extends the SQL language combined with many features that safely store and scale the most complicated data workloads.

Adaptable

Can cover a multiple of projects and involve multiple users. Multiple issues can be created and tracked and allows management at a central location.

Cost effective:

Great value for money. No overheads or implementation costs. No maintenance costs. Bugs dealt with free. Features developed for a nominal fee. Tailor made solutions for a minimum cost.

Transparency and accountability:

Use of charts and graphs to to visually display current and past project statistics as well as trends. Enables quick identification of trouble spots while also providing statistics to evaluate performance and aid in future planning.

Knowledge sharing

Participate in a multidisciplinary environment where knowledge is shared. Directly comment on issues or use the blog post to discuss wide ranging issues. Facilitates sharing of information & knowledge. Reduce the need for meetings that require everyone to be at the same place and at the same time.

Management Framework

An issues management framework is in place ensuring a comprehensive plan to deal with issues quickly and effectively.

For example:

  • specific people with specific expertise handle specific issues
  • issue are dealt with by both individuals and teams
  • upvotes determine which issues to prioritise
  • issues are communicated throughout the team through regular meetings, log checks, status updates emails, and so on.
